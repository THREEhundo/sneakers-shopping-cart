# Sneakers Shopping Cart

## Project Overview

A simple & clean shopping cart app.

- [Live](https://threehundo.github.io/sneakers-shopping-cart/)

### Description

- Clean UI & simple animations.
- Add & remove products from shopping cart.
- Responsive Design
